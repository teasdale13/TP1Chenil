Tout d'abord

Pour commencer, toute la navigation débute avec la barre d’outils située en haut de l’écran. Appuyez à gauche sur l’icône pour faire défiler les principales sections de l’application. Situé en haut à droite de l’écran, un autre menu avec des options destinées à l’administrateur.

Ajout d'éléments

Pour ajouter un chenil : Dans le menu de gauche sélectionnez l’option Chenil, ensuite vous avez simplement à appuyer sur le bouton + en bas à droite. Ensuite, un formulaire vous sera affiché et vous avez simplement à remplir les champs et à appuyer sur le bouton confirmer.

Pour ajouter un chien : Dans le menu de gauche sélectionnez l’option Tous les chiens, ensuite vous avez simplement à appuyer sur le bouton + en bas à droite. Ensuite, un formulaire vous sera affiché et vous avez simplement à remplir les champs et à appuyer sur le bouton confirmer.

Pour ajouter un chien a un chenil : Dans le menu de gauche sélectionnez l’option Chien par chenil, ensuite vous sélectionnez le chenil que vous désirez ajouter un chien. Ensuite vous avez simplement à appuyer sur le bouton + en bas à droite. Une liste vous sera affiché des chiens qui ne font pas déjà parti du chenil. Appuyez sur un chien de cette liste et il sera aussitôt ajouté au chenil.

Pour ajouter un poids : Dans le menu de gauche sélectionnez l’option Poids, ensuite vous sélectionnez le chenil. Sélectionnez le chien a qui vous voulez ajouter un poids et ensuite vous avez simplement à appuyer sur le bouton + en bas à droite. Un formulaire vous sera affiché et vous avez simplement à remplir les champs et à appuyer sur le bouton confirmer.

Pour ajouter une race : Dans le menu de gauche sélectionnez l’option Race, ensuite vous avez simplement à appuyer sur le bouton + en bas à droite. Deux options s'offrent à vous. Vous pouvez entrer vous même les informations ou dans la barre de recherche tappez une race et faites une recherche. Si une race se trouve sur la base de données en ligne en lien avec ce que vous avez saisi, une liste vous sera affichée. simplement la sélectionner et les champs se remplirons par eux-même. Ensuite simplement appuyer sur le bouton confirmer.

Suppression d'éléments

Pour Supprimer un chenil : Dans le menu de gauche sélectionnez l’option Chenil, ensuite vous avez simplement à glisser votre doigt de droite à gauche sur le chenil à supprimer. Une fenêtre s'ouvrira pour confirmer votre action.

Pour supprimer un chien : Dans le menu de gauche sélectionnez l’option Tous les chiens, ensuite vous avez simplement à glisser votre doigt de droite à gauche sur le chenil à supprimer. Une fenêtre s'ouvrira pour confirmer votre action.

Pour supprimer un chien associé a un chenil : Dans le menu de gauche sélectionnez l’option Chien par chenil, ensuite vous sélectionnez le chenil. Ensuite vous avez simplement à glisser votre doigt de droite à gauche sur le chien à supprimer. Une fenêtre s'ouvrira pour confirmer votre action.

Pour supprimer un poids : Dans le menu de gauche sélectionnez l’option Poids, ensuite vous sélectionnez le chenil. Sélectionnez le chien a qui vous voulez supprimer un poids et ensuite vous avez simplement à glisser votre doigt de droite à gauche sur le poids à supprimer. Une fenêtre s'ouvrira pour confirmer votre action.

Pour supprimer une race : Dans le menu de gauche sélectionnez l’option Race, ensuite vous avez simplement à glisser votre doigt de droite à gauche sur la race à supprimer. Une fenêtre s'ouvrira pour confirmer votre action.

Autres options

Pour sauvegarder votre base de données : Dans le menu de droite, sélectionnez l’option Setting, ensuite vous avez simplement appuyer sur le bouton Firebase et votre base de données est sauvegardée sur internet.

Pour visualiser un diagramme de l'âge des chiens : Dans le menu de droite, sélectionnez l’option statistique et un diagramme affichera le pourcentage des chiens selon leur âge.

Pour éviter la cosanguinité : Dans le menu de droite, sélectionnez l'option Généalogie.Ensuite sélectionnez le chien dont vous voulez faire accoupler et une liste apparaitra si des chiens sont cimpatible pour un accouplement.

Pour localiser les chenils : Dans le menu de gauche sélectionnez l’option Localisation, ensuite une carte affichera la position des chenils. à l'ouverture de la carte, un message s'affiche pour vous aviser de combien de chenils n'ont pu être afficher.

Pour scanner le colier (NFC) : Dans le menu de gauche sélectionnez l’option Scan, ensuite vous serez avisé si l'appareil supporte la détection NFC. Cette option vous ouvrira une fenêtre avec les informations du chien.
